## Node.js Example for integrating Rango-SDK-Basic for PSBT Transactions

### Requirements:

- rango-sdk-basic
- bitcoinjs-lib
- ecpair    
- tiny-secp256k1
- Node.js >= 20

### How to run the code?

Set up your wallet in index.ts, then run following commands in the root:

```sh
yarn
yarn build
cd /path/to/example/
node --import=tsx index.ts
```
